# 📊 E-Commerce Sales Dashboard (Power BI Project)

## 🚀 Project Overview

This project presents an interactive **E-Commerce Sales Dashboard** built using Power BI to analyze and visualize sales performance across multiple dimensions such as state, customer, category, and time.

The dashboard enables business stakeholders to monitor key performance indicators (KPIs), identify trends, and make data-driven decisions to improve profitability and operational efficiency.

---

## 🎯 Objectives

* Analyze overall sales, profit, and order quantity
* Track monthly profit trends and seasonal performance
* Identify top-performing states and customers
* Understand sales distribution by category and sub-category
* Analyze customer payment preferences

---

## 🛠️ Tools & Technologies

* **Power BI** (Data Visualization & Dashboarding)
* **Microsoft Excel / CSV** (Dataset)
* **DAX (Data Analysis Expressions)**
* **Power Query** (Data Cleaning & Transformation)

---

## 📂 Dataset Description

The dataset contains transactional e-commerce data including:

* Order Amount
* Profit
* Quantity
* Customer Name
* State
* Product Category & Sub-category
* Payment Mode
* Order Date

---

## 🔄 Project Workflow

### 1. Data Collection

* Imported raw e-commerce dataset into Power BI

### 2. Data Cleaning & Transformation

* Removed null and duplicate values
* Standardized column formats
* Performed data transformation using Power Query

### 3. Data Modeling

* Established relationships between tables
* Optimized data model for performance

### 4. DAX Calculations

Created important measures such as:

* Total Sales (Sum of Amount)
* Total Profit
* Total Quantity
* Average Order Value (AOV)

## 📊 Dashboard Features

### 🔑 Key Performance Indicators (KPIs)

* **Total Sales:** 438K
* **Total Quantity:** 5615
* **Total Profit:** 37K
* **Average Order Value:** 121K

---

### 📈 Visual Insights

#### 📅 Profit by Month

* Monthly profit trend visualization
* Identifies loss-making months (e.g., May, July, December)
* Highlights peak performance months

#### 🌍 Sales by State

* Maharashtra is the top-performing state
* Followed by Madhya Pradesh and Uttar Pradesh

#### 👥 Sales by Customer

* Top customers: Harivansh, Madhav, Madan Mohan
* Helps identify high-value customers

#### 💳 Payment Mode Analysis

* COD (44%) is the most preferred payment method
* Followed by UPI (21%) and Debit/Credit Cards

#### 🛍️ Category-wise Quantity

* Clothing dominates with 63% contribution
* Electronics and Furniture follow

#### 📦 Profit by Sub-Category

* Printers and Bookcases generate highest profit
* Tables and Accessories show lower profitability
